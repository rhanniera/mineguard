## 🔧 Quick Actions

### Change Report Status
1. Find report card
2. Click "Change Status" dropdown
3. Select: Pending → In Review → Resolved

### View Full Report
Click "View Details" on any report card
- See full description
- Read all comments
- Add new comments

### Delete Report
Click "Delete" button (cannot be undone!)
- Confirms before deletion
- Removes from system permanently

### Search Reports
Type in search box:
- Hazard title
- Location name
- Submitter name
- Description keywords

### Filter Reports
Use dropdowns:
- Status: All, Pending, In Review, Resolved
- Severity: All, Low, Medium, High, Critical

## 💾 Export Data

### Export as JSON
Button: "Export as JSON"
- Use: Data backup, system migration
- Format: Machine-readable JSON

### Export as CSV
Button: "Export as CSV"
- Use: Excel analysis, reports, printing
- Format: Spreadsheet compatible

## ⚡ Most Common Tasks

### Find Critical Issues
1. Filter: Severity → Critical
2. Sort by submission date
3. Review and update immediately

### Generate Monthly Report
1. Click "Export as CSV"
2. Open in Excel
3. Create pivot tables
4. Generate charts

### Track Resolution Progress
1. Filter: Status → In Review
2. Check each report
3. Add investigation notes
4. Update status as resolved

### Audit Specific Location
1. Search: Location name
2. Review all reports from location
3. Identify patterns
4. Plan remediation

## 📞 Emergency Response

**For Critical Hazard:**
1. Immediately change status to "In Review"
2. Add comment: "URGENT - [Your Action]"
3. Notify safety team
4. Update status when resolved
